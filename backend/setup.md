ADONISJS setup:

- move to ./VPWA/backend
- node -v
    to check if node.js is installed and using a good version

- npm init adonisjs@latest backend
    backend here being the name of the folder/project being initialized
- (y) proceed to install create-adonisjs@2.4.0

- starter kit => Inertia Starter Kit
    has capability of using frontend frameworks for Vue
- authentication guard => session
- database driver => postgres
- frontend adapter => vue
- server-side rendering with Inertia => (y)

the project should now be ready
- cd backend
- npm run dev
- open http://localhost:3333
